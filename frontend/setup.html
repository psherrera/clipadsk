Bienvenido a Clipadsk

Guía rápida para poner en marcha el transcriptor en una PC nueva.

1

Llave de IA (Gratis)

Clipadsk usa Groq para transcribir audios en segundos. Es gratis y muy rápido.

Obtener API Key open_in_new

Crea una cuenta, ve a "API Keys" y genera una nueva llave.

2

Dependencias

El archivo iniciar.bat intenta descargar todo automáticamente, pero si algo falla:

Descargar Python download Descargar FFmpeg download

info ¿Cómo empezar?

  1. Ejecuta iniciar.bat y espera a que instale las librerías.
  2. Cuando abra la ventana del transcriptor, ve a Configuración (icono de engranaje).
  3. Pega tu API Key de Groq y dale a guardar.
  4. ¡Listo! Ya puedes pegar links de YouTube o subir audios.
